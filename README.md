# Todopoderosos · Enciclopedia de Referencias

**12.490 referencias culturales** extraídas de 139 episodios del podcast [Todopoderosos](https://podcasts.apple.com/es/podcast/todopoderosos/id953023656).

Películas, libros, música, cómics, series, obras de teatro, videojuegos y arte mencionados en 300+ horas de conversación sobre cultura pop.

## 🔍 Funcionalidades

- **Búsqueda** por título, autor o nombre de episodio
- **Filtros por categoría** — cine, libros, música, cómics, TV, etc.
- **Vista por episodio** con número de referencias
- **Enlaces directos a Amazon** (con tag de afiliado `gragera-20`)
- Diseño oscuro responsive
